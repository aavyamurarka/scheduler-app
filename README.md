# scheduler-app

An AI scheduling assistant that reads the fixed events in your calendar (meetings, classes) and slots your flexible tasks (gym, groceries, deadlines) into the gaps in your day based on duration and importance.

## Status
In progress. Built with Cursor, Replit, and Claude. MVP core includes auth, tasks, calendar sync, and scheduling logic.

## Stack
Next.js · TypeScript · Supabase · Vercel
